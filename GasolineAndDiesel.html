    1. U.S. Energy Information Administration (EIA) API**: The EIA offers a free API providing access to a wide range of energy-related data, including historical gasoline prices. Their State Energy Data System (SEDS) contains estimates for average annual gasoline prices for each state dating back to 1970. To use the API, you'll need to register for a free API key.

  4. I am using Microsoft Power BI. How do I ingest an xml file?
  5. www.eia.gov/opendata
  1. Open Power BI Desktop.
  2. On the Home tab, click Get Data → More...
  3. In the Get Data window:
    • Search for or select XML.
    • Click Connect.
  4. In the file browser, navigate to your XML file, select it, then click Open.
  5. Power Query Editor will open, and you'll see the hierarchical structure of your XML.
  6. Navigate through the nodes to find the data you're interested in.
    • Expand tables by clicking the small expand icon (⏷) next to columns.
    • You might need to click through several nested tables/lists to get to the actual rows of data.
  7. Clean and transform the data as needed using Power Query tools:
    • Rename columns
    • Filter data
    • Change data types
    • Unpivot columns (if needed)
  8. Once ready, click Close & Apply in Power Query Editor to load the data into your Power BI model.
