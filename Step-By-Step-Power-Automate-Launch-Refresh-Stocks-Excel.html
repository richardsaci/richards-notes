Step By Step Power Automate Launch Refresh Stocks Excel


I was able to successfully launch and refresh stocks in Excel file using Power Automate. I am sharing steps below on how you can do this on your machine/login. If this doesnâ€™t work I can work with you to set this up. - Varun

  1. Create a new Power Automate flow.
  2. Copy the following code.
  3. Excel.LaunchExcel.LaunchAndOpenUnderExistingProcess Path: $'''C:\\PowerAutomate\\Stocks\\My-Tech-Stocks.xlsx''' Visible: True ReadOnly: False Instance=> ExcelInstance
  4. WAIT 120
  5. Excel.SaveExcel.Save Instance: ExcelInstance
  6. Excel.CloseExcel.CloseAndSave Instance: ExcelInstance
  7. Paste the code in new flow window.
  8. Select the empty space and hit CTRL+V
  9. Double click on the first steps to edit File Path.
  10. Click on browse button and select your Excel file that you want to refresh.
  11. Click SAVE and save the new flow.
  12. Test running the flow.
